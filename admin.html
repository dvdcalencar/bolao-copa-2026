Como funciona: arraste os arquivos .csv dos palpites que recebeu pelo WhatsApp. A página lê cada CSV, identifica o participante pelo campo User (PRONT-NOME) e cruza com a tabela participants do Supabase. Só monta SQL pra participantes que já estão cadastrados como pagos — quem não está, é ignorado (com aviso). Depois você copia o SQL gerado e cola no SQL Editor do Supabase. Se algum PRONT do CSV ainda não estiver cadastrado (ou estiver marcado como não pago), uma seção de cadastro de participantes aparece abaixo pra você gerar o SQL de inclusão deles.
